# ocaml-cli-buck2build-hello-world

## Description
POC compile ocaml programming with buck2 build tool.

## Tech stack
- buck2build
- ocaml
- clang
- lld

## Docker stack
- rust:latest

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
